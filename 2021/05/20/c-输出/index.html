c++ 输出

c++
学习

引入


你们可能并不知道c++输如输入数据的原理。在输入过程中，c++程序只是检查字节流，而并不检查字节的来源（可能来自于硬盘、键盘或者其他外设）。在处理输出方式时将独立于其去向，分为将流和输入去向的程序关联起来&将流与文件连接起来

通常使用缓冲区来提高输入输出的效率。从磁盘文件中读取字符需要大量的硬件活动，因此大量重复这个过程将会非常浪费时间。缓冲方法就是先将信息存储在缓冲区中，然后每次从缓冲区读取一个字节，这比直接从磁盘中读取要快很多

streambuf：为缓冲区提供了内存，提供了用于缓冲区的各种方法

ios_base：表示流的一般特征，如是否可读取、是二进制流还是文本流

ios：基于ios_base类，包含一个指向streambuf的指针成员


使用cout进行输出


可以通过输入重定向（<）和输出重定向（>），大多数的输入输出都连接着屏幕和键盘

cout代表的是标准输出是程序的输出的常用通道，标准错误流（由cerr&clog代表），用于输出错误信息


#include <iostream>
#include <cstdlib>
using namespace std;

int main(){
    int success;
    if(cin >> success){//这里否定的条件是程序异常终止导致值没有正常输入
        cout << "nice" << endl;
    }else{
        cerr << "error" << endl;
        exit(EXIT_FAILURE);
    }

    return 0;
}

<<也被重载，可以拼接使用（这个比较常用）

除此之外还提供了put()&write()方法，前者用于输出字符，后者用于输出字符串


#include <iostream>
using namespace std;

int main(){
    cout.put('A');
    cout << endl;
    cout.put(65).put(65.3);//通过.来连接输出
    //上面三个输出的结果是一样的，put会将数值当成asc码（double类型会直接类型转换）

    cout << endl << 'A' << endl;
    //在一些较老版本的c++中（c++2.0之前），会用int类型表示字符，这里结果就会是67（所以需要使用put）
    //但是现在都是直接用char类型来解释，两种方法都可以
    return 0;
}

个人感觉write方法使用的时候更像是for的范围循环，因为可以输出字符串中单个字符或者固定长度的字符

（要求必须是char类型的数组（指针），string无法通过转换从而使用）


#include <iostream>
#include <cstring>
using namespace std;

int main(){
    const char *str1 = "cxy,nice";
    long val = 987654321;
    int i;

    for(i = 1; i <= strlen(str1); i++){
        cout.write(str1, i) << endl;//一个表示需要的数组，一个表示输出的长度
    }
    cout.write((char *) &val, sizeof(val)) << endl;//该输出结果是一个乱码
    //将数值强制转换为char *，此时当作传输内存中存储的位表示
    //上述的long的四个字节将作为ASCII码独立输出，最后的结果是4个字符的组合，可能是乱码
    return 0;
}

c++输出可以使用flush来刷新缓存区，如果没有这个操作，程序在输入的时候将会先等待数值输入完毕，而不会输出其他提示信息

这个本身也是个函数，可以使用flush(cout)，当然c++重载了函数，使得使用更加方便


#include <iostream>
using namespace std;

int main(){
    cout << "looking for love" << flush;
    cout << "Waiting..." << endl;//endl和flush都是刷新缓存区，endl在刷新之后还会添加一个换行符
	flush(cout);
	
    return 0;
}

cout的精确设置


1.和c++也提供了更改输出格式的方法，使用dec、hex、oct更改进制（十进制、十六进制、八进制）

2.由于cout代替printf，所以c++使用width方法代替原c语言中的，这个方法可以设定字段宽度（注意：这个方法是一次性的，只能设定一个cout语句，结束之后恢复默认值）

3.在设置宽度之后，自然就会存在空余空间，c++默认使用space来填充

当然可以使用fill方法来使用字符来设置填充字符

4.需要控制浮点数显示位数的时候，就需要使用precision方法了（这个精度是包括前面的数的，0不算位（这个在c语言中有较大的篇幅提及））（c++的默认精度为6但是不会显示末尾的0）。跟width不同的是，这个东西是一直持续的，并不会默认恢复


#include <iostream>
using namespace std;

int main(){
    int num = 251;
    
    hex(cout);//也可以使用cout << hex;
    cout << num << endl;
    oct(cout);
    cout << num << endl;
    dec(cout);
    cout << num << endl;
    
    cout.fill('-');
    cout << "num";
    cout.width(5); 
    cout << num << endl;//只对单个插入符内部有效，如果是拼接的只会作用于第一部分

    double x = 0.1;
    cout.precision(1);
    cout << x << endl;

    return 0;
}

setf的各种使用


通常使用setf进行格式化输出，c语言中并没有这种规范化的定义（c++引入类的定义）

先介绍setf第一个版本，只是用一个参数

常量 含义
ios_base::boolalpha 输入和输出bool值，可以为true&false
ios_base::showbase 对于输出，使用c++基数前缀（0，0x）
ios_base::showpoint 输出末尾的小数点
ios_base::upercase 对于16进制的输出，使用大写字母，E表示法
ios_base::showpos 在正数前面加上+（只对十进制有效） 

#include <iostream>
using namespace std;

int main(){
    double x = 25.1;
    cout.setf(ios_base::showpoint);
    cout << x << endl;//补充剩下的0
    
    int temperature = 29;
    cout.setf(ios_base::showpos);
    cout.setf(ios_base::uppercase);
    cout.setf(ios_base::showbase);
    cout << temperature << endl;
    cout << hex << temperature << endl;//不会自动添加+号

    cout << dec << true << endl;//默认true输出值为1
    cout.setf(ios_base::boolalpha);
    cout << true << endl;//此时可以直接正常输出true

    return 0;
}

另一个版本的setf的使用在类和对象中有介绍过，使用两个参数

可以对cout进行更加精确的格式化控制


第二个参数 第一个参数 含义
ios_base::basefield ios_base::dec 使用10进制
ios_base::oct 使用8进制
ios_base::hex 使用16进制
ios_base::floatfield ios_base::fixed 使用定点计数法(即一般计数法)
ios_base::scientific 使用科学计数法
ios_base::adjustfield ios_base::left 左对齐
ios_base::right 右对齐
ios_base::internal 符合或前缀左对齐，值右对齐

虽然看起来比较麻烦，但是只要记住基本的归类，加上vscode的代码提示，还是很方便的就能写出来

调用的setf方法可以通过unsetf方法消除，如果将设置的setf位设置位1，unsetf位就是恢复为0


cout.setf(ios_base::showpoint);
cout.unsetf(ios_base::showpoint);
cout.setf(ios_base::boolalpha);
cout.unsetf(ios_base::boolalpha);

cout.setf(0, ios_base::floatfield);
//因为floatfield时是没有专门的显示标记的，所以可以直接使用0作为默认

当然使用setf方法并不是最好的格式化方式，c++提供了多个控制符，可以代替使用setf方法

这样之后可以直接使用简短的控制符，代替长串的setf代码

控制符 调用
boolalpha setf(ios_base::boolalpha)
noboolalpha unsetf(ios_base::noboolalpha)
showbase setf(ios_base::showbase)
noshowbase unsetf(ios_base::showbase)
showpos setf(ios_base::showpos)
noshowpos unsetf(ios_base::showpos)
uppercase setf(ios_base::uppercase)
nouppercase unsetf(ios_base::uppercase)
internal setf(ios_base::internal, ios_base::adjustfield)
left setf(ios_base::left, ios_base::adjustfield)
right setf(ios_base::right, ios_base::adjustfield)
dec setf(ios_base::dec, ios_base::basefield)
hex setf(ios_base::hex, ios_base::basefield)
oct setf(ios_base::oct, ios_base::basefield)
fixed setf(ios_base::fixed, ios_base::floatfield)
scientific setf(ios_base::scientific, ios_base::floatfield)

在头文件iomanip中，定义了一些其他的控制符，可以更加精确控制输出

常用的就是为setprecision()、setfill()、setw()分别对应设置精度，填充字符，宽度（只不过是控制符，可以直接在插入运算符中使用）


#include <iostream>
#include <iomanip>
#include <cmath>
using namespace std;

int main(){
    //如果要使用iostream中的表达式，应为cout.width(6),这样写太繁琐
    cout << setw(6) << "N" << setw(16) << "square result" << endl;
    cout << fixed;//使用一般计数法,在不使用一般计数法时，precision表示总位数；设置之后表示小数精度
    for(int n = 10; n <= 100; n += 10){
        cout << setw(6) << setfill('~') << n << setw(16) << setfill(' ')<< setprecision(4) << sqrt(double(n)) << endl;
    }

    return 0;
}
