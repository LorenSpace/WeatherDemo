迭代器优化


rend：返回程序的第一个值（和begin一样的返回值），类型不同，属于reverse_iterator（begin属于iterator）

rebegin：返回程序的最后一个值，类型同上

通过这两个迭代器加上输出迭代器，可以实现反向输出


#include <iostream>
#include <algorithm>
#include <iterator>
#include <vector>
using namespace std;

int main(){
    vector<int> v;
    ostream_iterator<int, char> ov(cout, " ");
    int num;

    cin >> num;
    while(num != -999){
        v.push_back(num);
        cin >> num;
    }

    sort(v.begin(), v.end());
    copy(v.begin(), v.end(), ov);
    cout << endl;
    copy(v.rbegin(), v.rend(), ov);//倒序输出，写法正常

    return 0;
}

在赋值进容器的时候，有时候会面临不知道所拷贝数据的长度或者单纯只是插入不覆盖原值的情况，这个时候就需要插入迭代器了

同时，c++也提供了三种插入类型的迭代器，插入将添加新的元素，而不会覆盖已有的元素

back_insert_iterator：将元素插入到容器的尾部

front_insert_iterator：将元素插入到容器的前端

insert_iterator：将元素插入构造函数参数的指定位置

在这里存在隐藏的限制，back_insert_iterator只允许在尾部进行快速插入的容器（快速插入算法），vector满足这种要求

front_insert_iterator智能允许在起始位置做时间固定插入的容器，vector就不能满足这种要求了，但是queue可以

虽然insert_iterator不用局限这种限制，但是写法会稍微麻烦一些


#include <iostream>
#include <vector>
#include <iterator>
using namespace std;
//插入的时候会自动申请内存
int main(){
   int s1[4] = {1, 2, 3, 4};
   int s2[2] = {5, 6};
   int s3[2] = {7, 8};
   vector<int> v(4);
   ostream_iterator<int, char> oit(cout, " ");

   copy(s1, s1 + 4, &v.front());
   copy(v.begin(), v.end(), oit);
   cout << endl;
   copy(s2, s2 + 2, back_insert_iterator<vector<int>>(v));//可以直接使用的快速插入的容器-vector
   //insert_iterator<vector<int>>(v, v.end()) 也可以写成这样
   copy(v.begin(), v.end(), oit);
   cout << endl;
   copy(s3, s3 + 2, insert_iterator<vector<int>>(v, v.begin()));//对v的特定位置进行插入
   copy(v.begin(), v.end(), oit);
   cout << endl;

   return 0;
}

容器


容器概念


原本是有11个容器queue、vector、map等等，在c++11中，又添加了几个新的

a.cbegin()：返回指向容器的第一个元素的const迭代器

a.cend()：返回超尾值const迭代器

（稍微解释固定时间和线性时间：拿取箱子举例子

固定时间即执行的任务和目标的个数没有关系，无论是100个箱子或是1000个箱子，消耗的时间是一样的

如果消耗的时间和箱子的数目成正比，这种任务就是线性时间复杂度）


序列


dequeue、list、queue、priority_queue、stack、vector、forward_list（c++11新增的容器）

这七种STL都属于容器，队列能够允许你在队尾添加元素，在队首删除元素

（array也被归类到序列容器，但是它并不满足序列的所有需求）

X a(n, t)：生命一个名为a的由n个t值组成的序列

a.insert(p, t)：将t插入到p的前面（注意是前面）

insert还有几个其他的重载insert(p, n, t)：插入n个t，insert(p, i, j)：向p前面插入i到j区间中的值（一般是数组或者其他容器）

a.erase(p)：删除p位置的元素。其重载a.erase(p, q)：删除p到q区间的元素

a.clear()：删除全部元素，相当于a.erase(begin(), end())

下面介绍不同序列的特点和用法

1.deque：表示双端队列，其实现类似于vector容器，支持随机访问

2.list：表示双向链表，除了第一个和最后一个元素意外，每个元素都与前后的元素相链接

这意味这，你可以双向遍历链表。list对任一位置的插入和删除都是固定时间，vector强调的是通过随机访问进行快速访问，list强调的是元素的快速插入和删除

和vector不同的是，list不支持数组表示法和随机访问。

merge(list<T, Alloc> &x)：将链表x和调用链表合并，合并前必须排序，合并完之后保存在调用链表里（合并之后，程序也会自动排序）

remove(const T &val)：删除链表中所有的val实例

sort()：使用<运算符进行正序排序

unique()：清除重复的元素

splice(iterator pos, lis<T, Alloc A> x)：将链表x的内容插入到pos前面，执行之后x将会清


#include <iostream>
#include <list>
#include <vector>
#include <iterator>
#include <algorithm>
using namespace std;

void print(int x){cout << x << " ";}

int main(){
    vector<int> v;
    int a[3] = {35, 35, 6};
    int num;
    cin >> num;
    while(num != -999){
        v.push_back(num);
        cin >> num;
    }

    list<int> l1;
    list<int> l2;
    //无法使用copy将值复制到list中
    l1.insert(l1.end(), v.begin(), v.end());
    l2.insert(l2.end(), a, a + 3);
    l1.sort();
    l2.sort();

    for_each(l1.begin(), l1.end(), print);
    cout << endl;
    for_each(l2.begin(), l2.end(), print);
    cout << endl;

    l1.splice(l1.begin(), l2);
    for_each(l1.begin(), l1.end(), print);
    cout << endl;
    for_each(l2.begin(), l2.end(), print);//此时的l2中为空
    cout << endl;
    l1.merge(l2);
    for_each(l1.begin(), l1.end(), print);
    cout << endl;
    l1.unique();
    for_each(l1.begin(), l1.end(), print);
    cout << endl;
   
    return 0;
}

3.forward_list：c++11新增的容器，只需要正向迭代器，不需要双向迭代器

相比于list更加简单&紧凑，但是对应的功能也就更少

4.queue：在头文件queue中，限制更多，不仅不允许随机访问队列元素，甚至不允许遍历队列

empty() const：如果队列为空，则返回true

push(const T &X)：在队尾插入x

pop()：删除队首元素


#include <iostream>
#include <queue>
using namespace std;

int main(){
    queue<int> q;
    int num;
    cin >> num;
    while(num != -999){
        q.push(num);
        cin >> num;
    }
    
    while(!q.empty()){
        cout << q.front() << " ";
        q.pop();
    }
    cout << endl;

    return 0;
}

关联容器


关联容器你可以理解为php中的键值对概念

最简单的就是set，键名和值的类型相同，每个键对应一个值（包含在set头文件中）

再往上进阶一点就是map，键名可以和值类型不同，不过每个键也是只能对应一个值（再来就是multimap，一个键可以对应多个值）（包含在map头文件中）

1.set：

set_union()：将两个set的并集传到另一个容器中

set_intersection()：查找两个set的交集

set_difference()：查找两个set的差


#include <iostream>
#include <set>
#include <string>
#include <vector>
#include <algorithm>
#include <iterator>
using namespace std;

const int N = 6;

int main(){
    string s1[N] = {"cxy", "about", "lgbt", "nice", "kpl", "no"};
    string s2[N] = {"customer", "lgbt", "arrange", "no", "please", "go"};
    set<string> A(s1, s1 + N);
    set<string> B(s2, s2 + N);
    vector<string> C;
    ostream_iterator<string, char> oit(cout, " ");//set的输出会自动排序

    cout << "A: ";
    copy(A.begin(), A.end(), oit);
    cout << endl;
    cout << "B: ";
    copy(B.begin(), B.end(), oit);
    cout << endl;
    cout << "A and B: ";
    set_union(A.begin(), A.end(), B.begin(), B.end(), oit);
    cout << endl;
    cout << "A in B: ";
    set_intersection(A.begin(), A.end(), B.begin(), B.end(), oit);
    cout << endl;
    cout << "A not B: ";
    set_difference(A.begin(), A.end(), B.begin(), B.end(), oit);
    cout << endl;

    cout << "C: ";
    set_union(A.begin(), A.end(), B.begin(), B.end(), insert_iterator<vector<string>>(C, C.begin()));
    //不能直接使用c.begin(),需要使用插入迭代器进行显示插入
    copy(C.begin(), C.end(), oit);
    cout << endl;
    return 0;
}

2.multimap：

multimap<int, string> does;：键名为int类型，储存的值为string类型

pair<class T, class U> item(213, "cxy");：通过pair模板类储存这种键名

（在pair对象中，可以使用first和second分别访问键名和值）

count(KeyType)：返回键名匹配的值的个数

equal_range()：返回键名相等的键值对（一般储存在pair中）


#include <iostream>
#include <string>
#include <map>
#include <iterator>
#include <algorithm>
using namespace std;

typedef int KeyType;
typedef string DataType;
typedef pair<KeyType, DataType> Pair;
typedef multimap<KeyType, DataType> Map;

int main(){

    Map codes;
    codes.insert(Pair(73042, "cxy"));
    codes.insert(Pair(73042, "ckt"));
    codes.insert(Pair(73041, "zpy"));
    codes.insert(Pair(73041, "lxq"));

    cout << "number in 73042: " << codes.count(73042) << endl;
    cout << "number\tname\t"<< endl;
    Map::iterator it;
    for(it = codes.begin(); it != codes.end(); it++){
        cout << (*it).first << "\t" << (*it).second << endl;//迭代器是指针的形式，调用对象方法的时候需要取地址
    }
    pair<Map::iterator, Map::iterator> range = codes.equal_range(73041);
    cout << "name in 73041: " << endl;
    for(it = range.first; it != range.second; it++){//这里不定义map的迭代器，直接使用auto i也没有问题
        cout << (*it).second << endl;
    }

    return 0;
}

函数对象引入


很多STL算法都是用函数对象——函数符（即在类中定义了operator()类）


#include <iostream>
using namespace std;

class Linear{
    private:
        double x;
        double y;
    
    public:
        Linear(double x0 = 1, double y0 = 1):x(x0), y(y0){cout << "created" << endl;}
        ~Linear(){cout << "closed" << endl;}
        double operator() (double x0){return x + y * x0;}
};

int main(){
    Linear l1;
    Linear l2(2.5, 10.0);
    double x1 = l1(12.5);
    double x2 = l2(0.4);
    cout << x1 << endl
    << x2 << endl;
    
    return 0;
}
