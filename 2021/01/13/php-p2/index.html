php一些好玩的地方


php中的一些语法比较简单随意

echo语句中使用单引号或双引号都没什么特别大的影响

echo语句中的几个变量可以由’.’号连接


<?php
$a='final';
$b='the game';
echo $a.' nice '.$b;
?>

输出结果

final nice the game

也可以换一种直接嵌套的方式


<?php
$a='final';
$b='the game';
echo $a.' nice '.$b.'<br>';
echo "$a nice $b";
?>

注意：此时echo中应当使用双引号

echo后面可以不加引号使用语句块表示输出结果

（Java与其类似，所以后端为什么不用Java来写（x）

当调用多个数据库有相同命名的类和对象的解决办法

php5.0加入了命名空间的概念，使其能够辨别不同的数据库（c++就开始了）


//创建一个function.php
<?php
namespace base;
$a='final';
$b='the game';
echo $a.' nice '.$b.'<br>';
echo "$a nice $b <br>";
class Hello{
    function sayHello(){
        echo 'Hello cxy'.'<br>';
    }
}
?>

//创建一个namespace.php
<?php
namespace hello;
class Hello{
    function sayHello(){
        echo 'Hello darling';
    }
}
?>

//主程序调用（use.php）
<?php
require 'function.php';
require 'namespace.php';
$a=new \base\Hello;
$a->sayHello();
$b=new \hello\Hello;
$b->sayHello();
?>

结果

final nice the game
final nice the game
Hello cxy
Hello darling

带参构造方法使用


首先说明一点，$this代表自身的对象

当变量用private修饰时，就不能直接调用了


//创建文件way.php
<?php
class Man{
    public function __construct($age,$name){
        $this->_age=$age;
        $this->_name=$name;
    }
    public function getAge(){
        return $this->_age;
    }
    public function getName(){
        return $this->_name;
    }
    private $_age,$_name;
}
?>

通过$_age , $_name 两个变量储存参数的值


//调用文件use.php
<?php
require 'way.php';
$a=new Man(10,'cxy');
echo $a->getAge();
echo $a->getName();
?>

输出结果

10cxy


成员方法和类方法


成员方法需要通过实例化进行访问

如同上面的public function getName()只能这么访问

类方法（静态方法,静态常量）

public static function getName()

const MAX_MAN_NUM=200

没有必要实例化进行使用只需要使用冒号来进行调用（本身已经进行过一次处理了）

主要是用来描述类的属性


//创建文件way.php
<?php
class Man{
    public function __construct($age,$name){
        $this->_age=$age;
        $this->_name=$name;
        Man::$man++;
        if(Man::$man>Man::man_max){//使用冒号进行调用
            throw new Exception("no room for it");//当超出某一条件时，跳出报错
        } 
    }
    public function getAge(){
        return $this->_age;
    }
    public function getName(){
        return $this->_name;
    }
    public static function sayHello(){
        echo 'hi!';
    }
    private $_age,$_name;
    private static $man=0;
    const man_max=200;//定义常量
}
?>//到这里你可能会觉得，php的写法更像java，php在这基础上引入某些更加方便地特性

//创建文件way.php
<?php
require 'way.php';
Man::sayHello();
for($i=0;$i<200;$i++){//将200换成Man::man_max也是可以的
    new Man(10,'cxy');
}
?>

页面成功显示出hi

如果将代码改成以下


<?php
require 'way.php';
Man::sayHello();
for($i=0;$i<=200;$i++){//多添加了一个等号
    new Man(10,'cxy');
}
?>

页面报错

Fatal error: Uncaught Exception: no room for it in D:\phpstudy_pro\WWW\www.wdnmd.com\way.php:8 Stack trace: #0 D:\phpstudy_pro\WWW\www.wdnmd.com\use.php(5): Man->__construct(10, ‘cxy’) #1 {main} thrown in D:\phpstudy_pro\WWW\www.wdnmd.com\way.php on line 8

类的继承和方法重写


如同孩子继承父亲，类是可以相互继承的

如果是子类调用父类，则会使用parent


//创建父类animal.php
<?php
class animal{
    public function __construct($age,$name,$sex){
        $this->_age=$age;
        $this->_name=$name;
        $this->_sex=$sex;
    }
    public function getAge(){
        return $this->_age;
    }
    public function getName(){
        return $this->_name;
    }
    public function getSex(){
        return $this->_sex;
    }
    public function sayHi(){
        echo $this->_name.' say hi';
    }
    private $_age,$_name,$_sex;
}
?>

//创建其子类dog.php
<?php
require 'animal.php';
class dog extends animal{
    public function __construct($age,$name,$sex){
        parent::__construct($age,$name,$sex);
    }
    public function sayHi(){//对父类的方法进行重写，定义相同名称的方法
        parent::sayHi();//如果想要保留原本的方法，即使用该语句
        echo '<br>';
        echo $this->getName().' say hi '.$this->getAge();
    }
}
?>

//使用例子main.php
<?php
require 'dog.php';
$a=new dog(10,'fwh','male');
$a->sayHi();
?>

结果显示如下

fwh say hi
fwh say hi 10**

json格式文件


首先先介绍json格式中常用的东西

格式如下


[1,2,5,7,"hello",[2,4,6]]
{"h":"hello","w":"world"}//这种格式称为键值对

语法


{"k":"v"}

”key“:”value“（简易的对象写法）

使用json_encode()来将php的数组转化为json形式

同理，使用json_decode()来讲json形式转化为php数组


<?php
$a=array('0','1','2',"hello",array(3,2,1));
$obj=json_encode($a);
echo $obj.'<br>';
$jsonStr='{"h":"Hello","w":"World","0":[1,2,3]}';//将json转化为数组
$jbo=json_decode($jsonStr);
print_r($jbo);
echo '<br>'.$jbo->h;//调用其中某一个键值对
?>

输出结果

[“0”,”1”,”2”,”hello”,[3,2,1]]
stdClass Object ( [h] => Hello [w] => World [0] => Array ( [0] => 1 [1] => 2 [2] => 3 ) )
Hello**

注意json格式中内容的里面的单引号


文件操作


无需像c语言一样先定义变量的类型（函数基本相同）

注意：当使用@fopen函数时，如果发生警告便不会输出

windows平台一般不会出现权限警告的问题


<?php
$f=@fopen('data','w+');
if($f==0){
    echo 'file error';
}else{
    fwrite($f,'Hello PHP');
    echo 'success';
}
fclose($f);
?>

可能是php协议的问题，导致文件无法被创建

除了类似于c语言的循环输出文件内容，php提供了一个内置函数

file_get_comtent(file name)

直接输出文件内容


图片操作


php图片主要可以用来生成验证码，压缩图片，限制上传大小，处理水印等

菜鸟教程有关内容


图片处理函数（自己查手册）


imagecrate()

里面传入长宽参数 


<?php
$a=imagecreate(300,400);
?>

但单单使用此函数会报错或者不显示

因为没有确定它的颜色


imagecolorallocate()

imagecolorallocatealpha()（调整图像颜色和透明度）

里面传入创建好的图片变量和rgb颜色参数，配合imagecreate()一起使用


<?php
$a=imagecreate(300,400);
imagecolorallocate($a,200,200,300);
?>

imagepng()

进行图片输出（以png格式）


<?php
$a=imagecreate(300,400);
imagecolorallocate($a,200,200,300);
header('Content-type:image/png');
imagepng($a);
?>

MIME TYPE的基本解释可以在我的其他博客中看到

可能考虑再搞个前端的typora什么的。只要不咕咕咕就行（bushi


imageellipse()

再原有图像上创建一个椭圆（圆）


<?php
$a=imagecreate(300,400);
imagecolorallocate($a,200,200,200);
header('Content-type:image/png');
imageellipse($a,200,200,60,60,imagecolorallocate($a,100,100,200));
imagepng($a);
?>

前两个参数是指圆的中心点的位置

后两个参数是指圆的宽和高，如果相同就是圆

最后则是圆的颜色


imagecreatefromjpeg()

直接调用文件中的图片显示再页面上

<?php
$image=imagecreatefromjpeg('preview.jpg');//函数中的参数为文件，这里我直接将图片复制到了本地
header('Content-type:image/jpeg');
imagejpeg($image);
//header('Content-type:image/png');
//imagepng($image);
//这样写函数同样也是正确的，图像的显示格式取绝你定的头
?>

图片水印处理


使用的函数为imagestring()

在图像指定地点添加一段字符串


<?php
$image=imagecreatefromjpeg('preview.jpg');
header('Content-type:image/jpeg');
imagestring($image,30,1,1,'two wives of cxy',imagecolorallocate($image,0,100,100));
imagejpeg($image);
?>

前三个参数分别是字体大小和它们的x，y坐标
