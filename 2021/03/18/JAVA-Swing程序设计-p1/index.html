JAVA Swing程序设计 p1

GUI（图形用户界面）为程序提供图形界面，原本的目的是为用户提供一个通用的GUI

但是Java中原来的AWT（抽象窗口工具箱）并没有达到这一点，Swing作为AWT的增强组件出现了


Swing概述


Swing主要拥有以下特点：

轻量级组件✔

可插入外观组件✔


Swing常用组件概述


组件名称 定义
JButton Swing按钮，按钮中可以使用文字和图片
JCheckBox 复选框组件
JComBox 下拉列表框，可以在下拉区域显示多个选项
JFrame 框架类
JDialog 对话框
JLabel 标签组件
JRadioButton 单选按钮
JList 在用户界面中显示一系列条目
JTextField 文本框
JPasswordField 密码框
JTextArea 文本区域
JOptionPane 一些对话区域

常用窗体


JFrame窗体


可以说，这个是Swing的基础，可以类比于html，建立一个基本的框架，之后才好在里面进行图像界面的完善

下面是一个标准的创建格式（未设置其他参数，只有标题）


JFrame jf = new JFrame(title);
Container container = jf.getContentPane();

getContentPane()方法将窗体转化为容器，并在其中进行操作

例如container.add(new JButton("按钮")); container.remove(new JButton("按钮"));

可以实现类似的基本操作


import javax.swing.*;
import java.awt.*;

public class JF_Practice extends JFrame {//类中需要继承JFrame，不然无法使用相应对象

    public void CreateFrame(String title){

        JFrame jf = new JFrame(title);//实例化对象，并给其一个标题
        Container container = jf.getContentPane();//获取容器

        JLabel jl = new JLabel("简单练习JFrame");//添加标签，使得窗体中显示出文字
        jl.setHorizontalAlignment(SwingConstants.CENTER);//设定标签上的文字居中显示
		
        //以下是设定框架中的基本参数
        container.add(jl);
        container.setBackground(Color.white);
        jf.setSize(200, 150);
        jf.setVisible(true);//设置窗体是否可视
        jf.setDefaultCloseOperation(WindowConstants.EXIT_ON_CLOSE);//设定关闭方式

    }

    public static void main(String[] args) {

        new JF_Practice().CreateFrame("创建一个JFrame");

    }

}

在Java的旧版本中，可以调用show()方法使得窗体可见，但是在新版jdk被舍去了

就从上述的代码来看，JFrame类可以构造有参或者无参的方法


创建窗体，需要配置一个关闭方式，Java提供了多种，这里写4个常用的

DO_NOTHING_ON_CLOSE 不关闭窗体

EXIT_ON_CLOSE 右上角按钮关闭,并且结束程序

HIDE_ON_CLOSE 窗体关闭之后只是隐藏，并不停止程序

DISPOSE_ON_CLOSE 调用任意已注册 WindowListener 的对象后自动隐藏并释放该窗体

程序默认为第三种关闭


JDialog窗体


JDialog的功能是从一个窗体中弹出另一个窗体，本质是另一种类型的窗体，与JFrame窗体类似

使用的时候也需要使用getContentPane()将窗体转化为容器

JDialog的创建和初始化和JFrame类似

public JDiaog() 创建一个没有标题和父窗体的对话框

public JDiaog() 创建一个指定标题、窗体和模式的对话框


import javax.swing.*;
import java.awt.*;
import java.awt.event.ActionEvent;

public class JD_Practice {

    public static void main(String[] args) {

        MyFrame myFrame = new MyFrame();

    }

}

class MyJDialog extends JDialog{

    public MyJDialog(MyFrame frame){

        //实例化一个JDialog对象，指定对话框的父窗体、窗体标题和类型
        super(frame, "第一个JDialog窗体", true);
        Container container = getContentPane();

        container.add(new JLabel("这是一个对话框"));
        setBounds(120, 120, 100, 100);//通过这个方法可以设定xy轴和大小

    }

}

class MyFrame extends JFrame{

    public MyFrame(){

        JFrame jFrame = new JFrame();
        Container container = jFrame.getContentPane();
        jFrame.setSize(500,500);
        jFrame.setBackground(Color.white);

        JLabel jLabel = new JLabel("这是一个JFrame窗体");
        jLabel.setHorizontalAlignment(SwingConstants.CENTER);
        container.add(jLabel);

        JButton jButton = new JButton("弹出对话框");
        jButton.setBounds(10, 10, 100, 50);

        jButton.addActionListener(new AbstractAction() {//为按钮添加鼠标单击事件

            @Override
            public void actionPerformed(ActionEvent e) {

                new MyJDialog(MyFrame.this).setVisible(true);

            }

        });

        container.add(jButton);

        jFrame.setVisible(true);
        jFrame.setDefaultCloseOperation(WindowConstants.EXIT_ON_CLOSE);

    }

}

标签组件与图标


Swing中显示文本或提示信息的方法是使用标签，支持文本和图标

public JLabel() 创建一个不带图标和文本的JLabel对象

public JLabel(Icon icon) 创建一个带图标的JLabel对象

public JLabel(String text, Icon icon, int aligment) 创建一个带文本、带图标的JLabel对象，并设定文字水平对齐方式


Swing中通过Icon接口来创建图标，可以在创建时给定图标的大小、颜色

在创建时，必须实现接口的三个方法

public int getIconHeight()

public int getIconWidth()

public void paintIcon(Component arg0, Graphics arg1, int arg2, int args)


import javax.swing.*;
import java.awt.*;

public class Icon_Practice {

    public static void main(String[] args) {

        DrawIcon drawIcon = new DrawIcon(15, 15);
        JLabel jLabel = new JLabel("测试", drawIcon, SwingConstants.CENTER);

        JFrame jFrame = new JFrame("Icon_Practice");
        Container container = jFrame.getContentPane();

        jFrame.setDefaultCloseOperation(WindowConstants.EXIT_ON_CLOSE);
        jFrame.setVisible(true);
        jFrame.setSize(500, 500);
        jFrame.setBackground(Color.white);

        container.add(jLabel);

    }
    
}

class DrawIcon implements Icon{

    private int height;
    private int width;

    //以下操作涵盖Icon接口中必需的方法
    @Override
    public int getIconHeight() {

        return this.height;

    }

    @Override
    public int getIconWidth() {

        return this.width;

    }

    public DrawIcon(int width, int height){

        this.width = width;
        this.height = height;

    }

    //Graphics包含了各种的图形
    @Override
    public void paintIcon(Component c, Graphics g, int x, int y) {

        g.fillOval(x, y, width, height);//调用方法画圆

    }

}

介绍的时候有提过，除了文字之外，JLabel还可以使用Icon

利用ImageIcon类根据现有图片创建图标

public ImageIcon() 创建一个通用对象，等到需要时可以使用setImage(Image image)

public ImageIcon(Image image, String description) 从图源创建图片，还可以为图片创建描述，可以使用getDescription()来获取对应信息

publice ImageIcon(URL url) 利用图像文件创建


import javax.swing.*;
import java.awt.*;
import java.net.URL;

public class MyImageIcon extends JFrame {//继承JFrame之后，下面写的方法就是对整个窗体操作了

    public MyImageIcon(){

        Container container = getContentPane();
        JLabel jLabel = new JLabel("这是一个JLabel", JLabel.CENTER);//在标签定义时规定样式

        URL url = MyImageIcon.class.getResource("./Pictures/1.jpg");//实例化一个URL实例，读取图片所在的位置
        Icon icon = new ImageIcon(url);
        jLabel.setIcon(icon);

        jLabel.setHorizontalAlignment(SwingConstants.CENTER);
        jLabel.setVisible(true);

        container.add(jLabel);
        setSize(250, 100);
        setVisible(true);
        setDefaultCloseOperation(WindowConstants.EXIT_ON_CLOSE);

    }

    public static void main(String[] args) {

        new MyImageIcon();

    }

}

布局管理


在这一块的内容就有点像前端的css了（子绝父相），使用布局管理比直接控制Swing组件的位置和大小要方便得多

主要包括流布局管理器、边界管理器&网络布局管理器


绝对布局


1.Container.setLayout(null) 取消布局管理器

2.Component.setBounds() 设置每个组件的大小与位置


流布局管理器


流布局就如起名，容器中的布局想”流“一样从左到右摆放组件，知道占据了这一行的所有空间

public FlowLayout()

public FlowLayout(int alignment, int horizGap, int vertGap)

alignment参数可以被赋予三个值之一

0 左对齐

1 居中

2 右对齐


import javax.swing.*;
import java.awt.*;

public class FlowLayoutPosition extends JFrame {

    public FlowLayoutPosition(){

        setTitle("流布局管理器");
        Container container = getContentPane();
        setLayout(new FlowLayout(1, 50, 50));//后两个参数主要设定高低间隔的大小

        for (int i = 0; i < 10; i ++){

            container.add(new JButton("button" + i));

        }

        setSize(1000, 1000);
        setVisible(true);
        setDefaultCloseOperation(WindowConstants.EXIT_ON_CLOSE);

    }

    public static void main(String[] args) {

        new FlowLayoutPosition();

    }

}

流布局中，如果改变整个窗体的大小，其中组件的摆放位置也会相应发生变化


边界布局管理器


边界（border），你可能会想到前端的盒式模型。margin、padding等等（虽然博主在前端考核时没去，但我也算半个前端人（x）

BorderLayout.NORTH 组件置于顶端

BorderLayout.SOUTH 组件置于底端

BorderLayout.EAST 组件右端

BorderLayout.WEST 组件置于左端

BorderLayout.CENTER 组件从中间开始填充


网格组布局管理器


由GridLayout类实现的布局管理

public GridLayout(int rows, int columns, int horizGap, int vertGap) 网格如其名，自然是设定行和列&每个布局布局的间隔

如果尝试改变窗体大小，组件的大小也会跟着改变

左上角的索引为0，此时组件左上角的点为起始点

gridx&girdy 设置组件起始点所在单元格的索引值

girdwidth&gridheight 设置组件占用网格的行数和列数

anchor 用来设置组件在显示区域的显示位置

fill 设置组件的填充方式，因为组件不一定填满整个区域

1.默认为NONE

2.设置为HORIZONTAL时，只调整组件水平方向的大小

3.设置为VERTICAL，只调整组件垂直方向的大小

4.BOTH，简而易懂，同时调整组件的水平方向&垂直方向大小

insets 设置组件四周与单元格边缘之间的最小距离Insets(int top, int left, int bottom, int right)设置组件的最小距离

ipadx&ipady 修改组件的首选大小，分别修改组件的宽度和高度

weightx&weighty 设置额外的空间，在窗体跳到足够大的情况下，避免出现窗体过大导致的大面积空白的情况


常用面板


JPanel面板


在JPanel中可以聚集一些组件来布局，面板本身也是容器的一种、

所以你单独设计面板，将组件放在面板中，然后将整体面板放在容器中


import javax.swing.*;
import java.awt.*;

public class JPanel_Practice extends JFrame {

    public JPanel_Practice(){

        Container container = getContentPane();
        container.setLayout(new GridLayout(2, 1, 10, 10));

        JPanel p1 = new JPanel(new GridLayout(1, 3, 10, 10));
        JPanel p2 = new JPanel(new GridLayout(1, 2, 10, 10));
        p1.add(new JButton("1"));
        p1.add(new JButton("2"));
        p1.add(new JButton("3"));
        p2.add(new JButton("4"));
        p2.add(new JButton("5"));
        container.add(p1);
        container.add(p2);

        setTitle("只是一个JPanel面板练习");
        setVisible(true);
        setSize(1000, 1000);
        setDefaultCloseOperation(WindowConstants.EXIT_ON_CLOSE);

    }

    public static void main(String[] args) {

        new JPanel_Practice();

    }

}

JScrollPane面板


在设置界面时，有可能会遇到在较小容器的窗体中显示一个较大部分内容的情况

带有滚动条的JScrollPane就适合这样的场景，但是该面板只能放置一个组件，并且不能使用布局管理器

当然，你可以先将组件放在JPane面板中，然后将其作为一个整体添加到JScrollPane面板中


import javax.swing.*;
import java.awt.*;

public class JScrollPane_Practice extends JFrame {

    public JScrollPane_Practice(){

        Container container = getContentPane();
        JTextArea jTextArea = new JTextArea(20, 50);

        JScrollPane jScrollPane = new JScrollPane(jTextArea);
        container.add(jScrollPane);

        setTitle("带滚动条的文字编译器");
        setSize(200, 200);
        setVisible(true);
        setDefaultCloseOperation(WindowConstants.EXIT_ON_CLOSE);

    }

    public static void main(String[] args) {

        new JScrollPane_Practice();

    }

}
