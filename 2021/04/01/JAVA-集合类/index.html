JAVA 集合类

在这里的集合，你可以看作c++中的栈

不过拜Java的接口所赐，集合写起来会更加的自然且方便，不需要再像c++那样了


collection接口


List集合和Set集合都继承了collection接口，所以可以使用对应的方法

在这里也是先简单了解一下集合类的使用


方法 功能描述
add() 将一个对象添加到集合中
remove 将对应的对象从集合中移除
isEmpty() 判断当前集合是否为空
iterator() 返回collection的元素上进行的迭代器，用于遍历集合中的对象
size() 获取集合中元素的个数 

import java.util.ArrayList;
import java.util.Collection;
import java.util.Iterator;

public class List_Practice {

    public static void main(String[] args) {

        Collection <String> list = new ArrayList<>();//实例化集合类对象
        list.add("陈鑫扬哥哥好帅");
        list.add("常陆茉子，我永远的老婆");
        list.add("回复术士这玩意竟然能播出来？");

        Iterator <String> iterator = list.iterator();//创建迭代器，返回值是一个对象
        
        while (iterator.hasNext()){//判断是否有下一个元素

            String str = iterator.next();//获取集合中的元素
            System.out.println(str);

        }

    }

}

List集合


list继承collection接口，因此可以使用该接口的所有方法，初次之外还有两个常用的方法

get(int index) 获得指定索引位置的元素（与数组相同，集合的索引也是从0开始）

set(int index, Object obj) 将集合中指定索引位置的对象修改为指定的对象

list的接口常用实现类也有两种

ArrayList类 实现了可变的数组，允许保存所有元素包括null，可以根据索引位置进行快速随机访问

LinkedList类 采用链表结构保存对象，这种结构的优点是便于向集合中插入和删除对象，这个时候的效率很高

此时我们可以知道，两个类互为优缺点


Set集合


Set集合中的对象不按特定的方式排序，只是简单的把对象放到集合中，所以这个类中不能有重复的对象

同时，这个Set集合也继承了Collection接口

Set接口常用的实现类有HashSet类与TreeSet类

HashSet 由哈希表支持，不保证Set的迭代顺序，特别是不保证该顺序保持不变。此类允许使用null元素

TreeSet 不仅实现了Set接口，还实现了java.util.SortedSet接口，因此实现在遍历集合时按照自然顺序递增排序


方法 功能描述
first() 返回集合中的第一个元素
last() 返回集合中的最后一个元素
comparator() 返回对此Set中的元素进行排序的比较器，如果使用自然排序，则返回null
headSet() 返回一个新的Set集合，新集合是括号中对象之前的所有对象（不包括本身）
subSet() 返回在两个对象之间的所有对象（不包括本身）
tailSet() 返回之后的对象（包括本身） 

import java.util.Iterator;
import java.util.TreeSet;

public class Set_Practice{

    public static void main(String[] args) {

        StudentMessage message1 = new StudentMessage("陈鑫扬", 1);
        StudentMessage message2 = new StudentMessage("范文豪", 2);
        StudentMessage message3 = new StudentMessage("陆晓琦", 3);
        StudentMessage message4 = new StudentMessage("张运培", 4);

        TreeSet <StudentMessage> set = new TreeSet<>();
        set.add(message1);
        set.add(message2);
        set.add(message3);
        set.add(message4);

        Iterator iterator = set.iterator();//Set集合中所有元素的迭代器

        System.out.println("输出集合中的所有元素");
        while (iterator.hasNext()){

            StudentMessage stu = (StudentMessage) iterator.next();
            System.out.println("姓名：" + stu.getName() + "  id：" + stu.getId());

        }

        iterator = set.headSet(message2).iterator();//截取前面的对象
        System.out.println("输出集合前面的所有元素");
        while (iterator.hasNext()){

            StudentMessage stu = (StudentMessage) iterator.next();
            System.out.println("姓名：" + stu.getName() + "  id：" + stu.getId());

        }

        iterator = set.subSet(message2, message3).iterator();
        System.out.println("输出集合中间的所有元素");
        while (iterator.hasNext()){

            StudentMessage stu = (StudentMessage) iterator.next();
            System.out.println("姓名：" + stu.getName() + "  id：" + stu.getId());

        }

    }

}

class StudentMessage implements Comparable<Object>{

    private String name;
    private int id;

    public StudentMessage(String name, int id){

        this.name = name;
        this.id = id;

    }

    @Override
    public int compareTo(Object o) {//规定比较的方式

        StudentMessage message = (StudentMessage) o;
        return Integer.compare(id, message.id);
        //Integer类中的一个方法，比较前后两个整形书的大小，大于返回1，等于返回0，小于返回-1

    }

    public void setName(String name) {

        this.name = name;

    }

    public String getName() {

        return name;

    }

    public void setId(int id) {

        this.id = id;

    }

    public int getId() {

        return id;

    }

}

Map集合


Map集合比较特殊，其没有继承Collection接口，使用的是键值对（这个在我php中讲过）


方法 功能描述
put() 向集合中添加对应的键值对
containsKey() 判断存在对应的键名
containsValue() 判断存在对应的值
get() 如果存在对应的key，就返回对象的值
keySet() 返回所有key对象形成的Set集合
values() 返回该集合中所有值对象形成的Collection集合 

package Collection_Practice;

import java.util.*;

public class Map_Practice {

    public static void main(String[] args) {

        Map<String, String> map = new HashMap<>();
        map.put("01", "cxy");
        map.put("02", "fwh");

        Set <String> set = map.keySet();
        Collection <String> collection = map.values();
        Iterator <String> iterator = set.iterator();//加入<String>可以规定迭代器的范围

        System.out.println("Map中的键名");
        while (iterator.hasNext()){

            System.out.println(iterator.next());

        }

        iterator = collection.iterator();
        System.out.println("Map中的键值");
        while (iterator.hasNext()){

            System.out.println(iterator.next());

        }

    }

}
